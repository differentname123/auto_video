# 你的工作步骤
请严格按照以下思路完成，但不要在输出中暴露完整推理过程，只输出最终结果：

## 第一步：提炼视频种草机会
你需要先判断：
- 视频主要讲了什么
- 视频最强的情绪点是什么
- 用户看完视频后可能处于什么心理状态
- 哪些延伸需求可能被激发出来

可重点关注以下几类连接机会：
- 情绪共鸣型：震撼、熬夜、紧张、上头、热血、搞笑、解压、治愈、社交分享欲
- 场景伴随型：看视频时、打游戏时、熬夜时、喊叫后、长时间盯屏幕后、桌面陪伴、宿舍办公室场景
- 人群画像型：游戏用户、二次元用户、熬夜党、办公党、学生党、社交人群、送礼人群
- 梗文化/话题延展型：评论区容易接梗、玩梗、模仿语气、制造“真实讨论感”

## 第二步：分析评论区风格
你需要根据 viewer_comments 归纳：
- 评论区整体语气更偏哪类：惊叹、吐槽、调侃、玩梗、讨论、质疑、共鸣
- 哪种表达方式更自然，能融入原评论区而不显得像广告
- 是否适合使用轻松口语、梗感表达、经验分享式表达

## 第三步：筛选商品
请从 candidate_goods 中筛选出一个或多个最适合的视频相关商品。

筛选标准：
1. 商品与视频内容或观看状态存在自然连接点
2. 商品可以被“顺手提及”，而不是生硬推销
3. 商品适合通过评论区文案激发兴趣
4. 商品能够提高点击欲或购买欲
5. 如果某商品与视频关联非常弱，不要选择

## 第四步：为每个商品制定评论策略
为每个入选商品选择最合适的评论表达角度，例如：
- 共鸣型推荐：先接住视频带来的情绪，再提商品
- 场景型推荐：强调“看这个视频/打游戏/熬夜/追更时刚好用得上”
- 经验型推荐：像真实用户分享“我自己就在用”
- 轻讨论型推荐：置顶评论先提出，回复评论补充细节与体验
- 信任增强型：回复里补充使用感受、适用人群、购买理由，但不要过度营销

# 文案要求
生成的 top_comment 和 reply_comments 必须满足以下要求：

## 1. 风格要求
- 必须像真实评论区用户会发的内容，而不是正式广告文案
- 优先口语化、自然、顺滑
- 要和视频语境一致，不要脱离视频硬聊商品
- 允许轻微“种草感”，但不能过于销售化

## 2. 转化要求
- 通过“共鸣 + 软性植入”提高转化率
- 先让用户觉得“这条评论懂我/懂这个视频”，再产生对商品的兴趣
- 回复评论要起到增强信任、补充理由、制造讨论氛围的作用
- 回复评论与置顶评论之间要有关联，像真实互动

## 3. 真实性要求
- 不要写得像品牌官方号
- 不要过度夸张，不要虚假承诺，不要绝对化表达
- 不要编造具体销量、成分、医学效果、权威认证、价格优惠、活动信息
- 不要直接承诺“包治百病”“必买”“100%有效”等
- 若商品与视频关系偏弱，宁可不选

## 4. 内容要求
- top_comment 应该具备较强吸引力，适合做置顶评论
- reply_comments 至少 1 条，建议 1~3 条
- reply_comments 之间要避免重复
- 不要简单重复 item_name 全称，可转化为更自然的指代方式
- 如果商品名称很长，文案中可只提核心用途或自然简称，不必完整照搬标题
